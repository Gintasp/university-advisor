## Project feature roadmap
This file describes various features to be implemented into the project by splitting them into 3 project stages: `core`, `minimum`, `medium` and `maximum`.

#### Features:
1. Core: 
	* Lists of universities, lectures, study programs and courses
	* Written reviews of universities/lectures
	* Scores for universities/lectures
2. Minimum:
	* Login/sign-up system
	* Database/files for storing info of user profiles/universities/lectures
	* Various statistics and comparison of universities/lectures based on anonymous alumni data (average salary, satisfaction, studying experience etc.)
3. Medium:
	* Migration to ASP .NET MVC
	* Mode for international students: sort out courses that are in English
	* OAuth
	* Google Maps integration (display distances to dormitories in faculty page)
	* Dark Mode	
4. Maximum:
	* Forum for various discussions
	* Possibility to view and compare your studies path
	* Some kind of report/vote system for worst lecturers, automatic alert for someone at university to look in lecturers' work and make some decision.	
	* Ability for students to post their exams/material for studying
